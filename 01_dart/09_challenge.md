# **09_challenge（条件分岐・関数）**

## **問題1**

XXXの部分に記号を入れ、文字を出力させよう  

```dart
void main() {
  
  var number = 21;
  if (number XXX 20){
    print('numberは20よりも大きい');
  }

}
```

```
>> numberは20よりも大きい
```
